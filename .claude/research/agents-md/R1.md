# R1 — root-core inventory

> Scope: `src/`, `tests/`, root configs, root markdowns, entry points.
> Storage class: `git` = tracked, `gitignored` = ignored, `hybrid` = mixed.
> Backlinks: `[[path]]` for AGENTS.md cross-references; `[path](path)` for code files.

---

## Entry Points

| File | Role | Storage |
|------|------|---------|
| [`mcp_server.py`](mcp_server.py) | Thin stdio MCP proxy (~20 MB). Forwards all tool calls via HTTP to `daemon.py` on localhost:8742. Auto-starts daemon if not running. | git |
| [`daemon.py`](daemon.py) | Persistent HTTP server holding ML models in memory (~1.4 GB total). Serves `/tool/<name>`, `/health`, `/admin/unload`, `/admin/shutdown`. | git |
| [`cli.py`](cli.py) | Bash-accessible HTTP client for daemon tools. Enables sub-agents without MCP access to call search, analyze-task, trace-impact, etc. | git |
