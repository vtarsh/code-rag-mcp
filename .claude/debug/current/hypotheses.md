- result: excluded (cross-evidence from prior leg-swap) — two-tower v13 deploy (memory `project_two_tower_v13_landed.md`) swapped the vector leg from CodeRankEmbed to nomic-embed-text-v1.5 for doc-intent and measured "Benchmarks flat vs v8" — if one leg materially dominated recall, a leg swap would have moved the bench. - test: instrument `hybrid_search` with per-query stage logs — `gt_in_fts150`, `gt_in_vec50`, `gt_in_rrf_top200`, `gt_in_rerank_input`, `gt_in_top10_post_rerank`. Aggregate over jira n=908. Drop-off curve isolates the lossiest transition.
